# Mocha Sinon
Samples using `Sinon` with Mocha (uses Chai internally)

- you can run tests
$ npm test

- no ci yet (will be adding circle and travis at some point)

- mocha
- sinon
- chai (expect, should, assert)	
- todo (creates a markdown file of all todo references in src)
